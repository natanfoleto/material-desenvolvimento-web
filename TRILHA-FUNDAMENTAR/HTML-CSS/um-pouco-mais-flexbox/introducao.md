# Um pouco mais sobre Flexbox

Vimos nas aulas anteriores o Flexbox

* display: flex;
* flex-direction: column | row;
* gap: 10px;
* align-items: center;
* justify-content: space-arround;

## Aqui a baixo teremos mais algumas muito úteis

* flex-wrap
  Display flex em linhas

* flex-basis
  Tamanho dos elementos no diplay flex

* flex-grow
  O crescimento do item dentro do container em relação aos espaços vazios

* flex-shrink
  O encolhimento do item dentro do container
